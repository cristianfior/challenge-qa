##### Cadastro Pessoal
    RN005.9: Os campos  Nome, Sobrenome, Nome social, Cidade, Estado, País NÃO devem aceitar valores numéricos;
    RN005.10: Os campos  Data de nascimento, Celular, Telefone, CEP, SÓ devem aceitar valores numéricos;
    RN005.11: Os campos Celular, Telefone devem ter validação contra valores repetidos. (ex. (11) 11111-1111);
    RN005.12: O campo Data de nascimento não deve aceitar ano maior que 2008 ou menor que 1906;
    RN005.13: O campo Email deve ter validação regex de email;
    RN005.14: O campo CEP deve possuir integração com API de busca de cep para validação e automação no processo de preenchimento de endereço;
    RN005.15: O campo Endereço deve ser destrinchado nos campos Logradouro e Numero para persistência de dados no banco de dados. Protótipo: evidencias/melhorias/rn005-15.png
